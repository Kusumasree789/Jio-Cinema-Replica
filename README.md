# Jio Cinema Replica

A simple project replicating the user interface and functionality of the Jio Cinema platform. This project demonstrates how to build a movie streaming interface using modern web technologies such as HTML, CSS, and JavaScript.

## Features

- **Movie Listing**: Displays a list of movies, mimicking a streaming platform interface.
- **Interactive UI**: Provides a user-friendly interface to browse and explore movies.
- **Responsive Design**: Fully responsive layout for mobile and desktop views.

## Tech Stack

- **CSS**: For styling and layout design.
- **JavaScript**: For dynamic content rendering and interactions.
- **JSON**: For storing movie data.
